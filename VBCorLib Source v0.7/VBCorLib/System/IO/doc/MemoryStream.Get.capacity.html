MemoryStream: capacity (get)

capacity

Returns the current capacity of the internal stream buffer. 

 Public Property Get capacity ( ) As Long

Return Values

Long -  The number of bytes the internal array contains.

Remarks

If the the capacity is reached while writing, then the capacity will be expanded, unless the underlying byte array is user-supplied. In this case, an exception will be thrown if the capacity is changed.
